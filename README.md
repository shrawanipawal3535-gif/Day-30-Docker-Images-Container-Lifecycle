# Day-30-Docker-Images-Container-Lifecycle
Docker images are templates used to create containers. Containers follow a lifecycle such as create, start, stop, and remove. Images are built using layers which help reduce storage. Commands like docker run, docker ps, and docker inspect are used to manage containers.
